					Any from this small mods can be removed without any restricts and conflicts
											But better do it before start new save.

											===== Description =====

- AlternatorForAll - add small alternators to all engines, except boosters and which already have - MM patch

- ChuteTweak - tweaking of parachutes drag and trigger heigths - MM patch

- CommandInventory - build-in KIS storage for all crewed parts, only if you have KIS installed - MM patch

- DecouplerCrossfeed - adds abiliity to toggle crossfeed in any decoupler - MM patch // need test with 1.2.1, may be in stock

- ElectricCost - no more free electricCharge. But it still cheap. - MM patch

- EngineResponseTime - add small (not like jeeets) response time for all engines - MM patch

- FSFuelSwitch - allow changing fuel type in LF+OX or MP tanks, including non-stock. Required Firesplitter - MM patch

- NervPlumeFix - small visual improvements for LV-N engine exhaust, move it closer to bell - MM patch

- SmallGimbal - add small gimballing (0.4 degree only) to all engines, except boosters and which already have - MM patch
